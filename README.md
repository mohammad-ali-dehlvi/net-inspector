# Net Inspector
- Created on `Express`, `React`, `Playwright`, `Socket`, and `Typescript`. 
- It records the session video of the browser with its network requests

# How to use
- Take pull from `main` branch
- Then Run
```
npm install
```
  This will install all the dependencies
- Then Run
```
npm start
```
  This will start the `Frontend` as well as the `Backend` server.

**Note**

Frontend PORT: `5173`, URL: [http://localhost:5173](http://localhost:5173)

Backend PORT: `8000`, URL: [http://localhost:8000](http://localhost:8000)
